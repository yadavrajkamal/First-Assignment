HTML Tags

  1. <html>: The <html> tag represents the root of an HTML document. All other HTML elements must be descendants of this element.
  2. <head>: The <head> tag contains meta-information about the HTML document, such as its title and links to scripts and stylesheets.
  3. <title>: The <title> tag defines the title of the document, which is shown in the browser's title bar or tab.
  4. <body>: The <body> tag represents the content of an HTML document. It contains all the content that is displayed in the browser.
  5. <h1> to <h6>: The <h1> to <h6> tags define HTML headings. <h1> defines the most important heading, and <h6> defines the least important heading.
  6. <p>: The <p> tag defines a paragraph of text.
  7. <a>: The <a> tag defines a hyperlink, which is used to link from one page to another. The href attribute specifies the URL of the page the link goes to.
  8. <img>: The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image.
  9. <ul>: The <ul> tag defines an unordered list.
  10. <ol>: The <ol> tag defines an ordered list.
  11. <li>: The <li> tag defines a list item and is used inside <ul> or <ol> tags.
  12. <div>: The <div> tag defines a division or a section in an HTML document. It is used as a container for other HTML elements.
  13. <span>: The <span> tag is used to group inline-elements in a document. It provides a way to style parts of the text or content.
  14. <form>: The <form> tag is used to create an HTML form for user input.
  15. <input>: The <input> tag specifies an input field where the user can enter data.
  16. <button>: The <button> tag defines a clickable button.
  17. <table>: The <table> tag defines a table.
  18. <tr>: The <tr> tag defines a row in a table.
  19. <td>: The <td> tag defines a cell in a table.
  20. <th>: The <th> tag defines a header cell in a table.
  21. <style>: The <style> tag is used to define CSS styles within an HTML document.
  22. <link>: The <link> tag defines a relationship between the current document and an external resource. It is most commonly used to link to stylesheets.
  23. <script>: The <script> tag is used to embed or reference executable code, typically JavaScript.
  24. <meta>: The <meta> tag provides metadata about the HTML document. It is used within the <head> section.
